  • handleAPIError(error: any): void
    • Defined in app/services/TaskManagerService.ts:319

    Handles a GitHub API error. If the error code is 404, the task is invalid, it will be removed. If the error code is 403, the API is unavailable at this moment, we will retry in one minute. If the error code is other (like rate limit exceed) we will retry when the API tells to retry.
