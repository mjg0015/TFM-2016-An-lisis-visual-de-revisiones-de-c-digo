Pedir un repositorio a GitHub

La petición de repositorios está deshabilitada.

La funcionalidad de petición de repositorios software, aunque es una de las principales características de esta aplicación, supone el almacenamiento de grandes cantidades de datos en nuestro servidor.
Permitir la libre obtención de nuevos repositorios saturaría el pequeño servidor de base de datos que estamos utilizando para esta demo.
